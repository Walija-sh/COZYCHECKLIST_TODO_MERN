## Features

- Create, update, and delete tasks
- Mark tasks as completed or pending
- Filter tasks by status (completed / pending)
- Filter tasks by priority (High, Medium, Low)
- Sort tasks by priority with pending tasks first
- Optimistic UI updates for fast and smooth experience
- User authentication (Sign Up / Sign In)
- User profile page
- Responsive design for desktop and mobile
- 404 page for unmatched routes

---

## Tech Stack

**Frontend:**

- React 18
- React Router DOM
- Axios
- Tailwind CSS
- React Toastify
- Lucide React Icons

**Backend:**

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT authentication
- CORS & Cookie-based auth

**Deployment:**

- Frontend: Vercel
- Backend: Vercel (Serverless Node API)
